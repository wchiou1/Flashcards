# flashcards
Converts text document to interactive flash cards to aid in studying

Key symbols
>  :Defines a group
-  :Defines a card(first line will be the question, all other lines after will be the answer)

Example:

>Group 1

-Question 1:
Answer 1
-Question 2:
Answer 2.a
Answer 2.b

>Group 2
-Lone Question:
Lone Answer
